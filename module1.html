常用指令

使用说明：以下指令均为服务器控制台指令（无需斜杠），若在游戏内使用需要在指令前添加 / 符号。

死亡不掉落

gamerule keepInventory true
开启后玩家死亡时不会掉落物品和经验

禁用昼夜更替

gamerule doDaylightCycle false
固定游戏时间，不会出现白天和黑夜的交替

禁用自然生命恢复

gamerule naturalRegeneration false
玩家无法通过饥饿值恢复生命值，需要通过其他方式回血

禁用生物生成

gamerule doMobSpawning false
禁止游戏中自然生成怪物和动物

禁用方块掉落

gamerule doTileDrops false
破坏方块时不会掉落物品（如挖矿不会掉矿石）

开启雷电天气

weather thunder
将天气切换为雷暴天气，持续时间默认30秒
重要提醒：使用指令前请确保拥有管理员权限(op)，部分指令需要重启服务器才能生效。
